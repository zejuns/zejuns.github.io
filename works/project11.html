Color Network Model

The color network model can use k-means clustering to images and create color pie charts. These pie charts are then used to develop a color network model that explores the color combinations and patterns within a specific type of image. After the color network model is formed, the program allows you to select a color and provides color combination suggestions. You can also input a new image, and the program will replace the colors in the image based on the suggested color combinations.
